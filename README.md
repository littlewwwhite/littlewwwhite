### What I Work On

- **Agent Runtime / Harness Engineering**: treating prompts, tools, memory, sandbox, feedback loops, and evaluation as one executable environment.
- **Retrieval / Knowledge Systems**: reasoning-oriented RAG, knowledge graph QA, document indexing, code search, and structure-aware retrieval.
- **AI Creative Infrastructure**: experimenting with cinematic production OS, world models, image/video/3D asset pipelines, and game/film workflow automation.
- **Developer Tooling**: small tools that reduce friction in AI coding, model operations, and day-to-day engineering.

### Operating Principles

- Keep source content lossless; structure can be added later, but observations should not be destroyed.
- Prefer long half-life interfaces over short half-life patches.
- Build harnesses, feedback loops, and evaluations before adding more prompt rules.
- Make systems observable enough that failures can be studied instead of guessed.
